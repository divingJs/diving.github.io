.addClass()
Te permite agregar una o varias clases a un elemento del dom
<style type="text/css">
	.yellow{
		background-color:yellow;
	}
	.green{
		background-color:green;
	}
</style>
<p>
	Este es un estilo yellow
</p>
<p id="green">
	Este es un estilo green
</p>

<script type="text/javascript">
	dvn('p').addClass('yellow');
	dvn('#green').addClass('green');
</script>

Este es un estilo yellow

Esete es un estilo green
