Propperties

.addClass()

Permite agregar una o varias clases a un elemento.

.append()

Agrega un objeto diving a un elmento, incluye los atributos que este contenga.

.attr()

Muestra los atributos de un elemento o bien permite agregar atributos a un elemento ya existente.

.center()

Centra el elemento en la pantalla.

.css()

Agrega estilo directamente al elemnto ( element.style )

.data()

Permite guardar un objeto en cache asociado a un elemento del DOM y reutilizar posteriormente.

.empty()

Limpia el objeto diving

.find()

Devuelve un array con los elementos encontrados mediante el parametro que se pasa.

.hasClass()

Devuelve true o false si tiene o no la clase que se pasa como parametro.

.hide()

Oculta el elemento.

.parent()

Devuelve el elemento padre en el que se encuentra el elemento que se esta parentando.

.prepend()

Inserta el objeto diving antes del contenido que este tenga.

.remove()

Remueve el elemento al igual que los elementos contenidos dentro de el.

.removeAttr()

Remueve el attributo que se pasa como parametro.

.removeClass()

Remueve la clase que se pasa como parametro.

.show()

Vuelve visible el elemento.

.text()

Obtiene o reemplaza el texto ( contenido ) de un elemento.

.toggleClass()

Intercambia la clase actual del elemento con la que se pasa como parametro.

.trigger()

Dispara un trigger sobre el elemento seleccionado.
Implementacion
Comienze a utilizar diving de una manera bastante sensilla y facil de usar, manejando unicamente el apartado de js, o bien, si lo que requiere un diseño rapido y de alta calidad.
Agase de nuestro apartado CSS, que le brindara una alta variedad de estilo y diseño a la altura de sus necesidades.
Sin dejar de mensionar que contamos con mas de 5000 iconos que divingJS pone a tu disposición.
El uso esclusivo de nuestra libreria JS, brindara un sin fin de funciones para un facil manejo de los elmentos dentro del dom. 
<style type="text/javascript" src="lib/diving-3.0.1/js/diving-core-3.0.1.js"></style>
por otro lado, divingJs cuenta con un estilo atractivo para internet y sus usuarios. 
<link rel="stylesheet" type="text/css" href="lib/diving-3.0.1/css/diving-3.0.1.css" />
<style type="text/javascript" src="lib/diving-3.0.1/js/diving-core-3.0.1.js"></style>
Methods

.addPx()

.draggable()

Construye a un objeto del DOM en un elmento draggable.

.each()

Recorre un arreglo devolviendo un callback( posicion, elemento )

.extend()

Permite extender las propiedades de un objeto, o bien en algunos casos sobreescribirlas.

.find()

Busca un elemento dentro del DOM

.isFunction()

Evaluar si una variable es o no funcion.

.map()

Mapea un objeto

.template()

Realiza un template de algun elemento del dom
